## Description

[Goal-Achiever](https://github.com/maxts/goal-achiever) Sample project for use as skeleton using Nestjs, sqlite3, React w/ Typescript & Material Design.

## Installation

```bash
$ npm install
```

## Running the app

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```

## Test

```bash
# unit tests
$ npm run test

# e2e tests
$ npm run test:e2e

# test coverage
$ npm run test:cov
```
